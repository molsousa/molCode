# Checklist

## Modo de insercao

#### Teclas a adicionar:

[X] PageUp: Mover cursor ao inicio da linha atual.

[X] PageDown: Mover cursor ao final da linha atual.

[X] Home: Mover cursor ao inicio do arquivo.

[X] End: Mover cursor ao final do arquivo.

#### Atribuicoes de teclas:

[X] Definição de todas as constantes necessárias.

[X] Ctrl + X: Para copiar uma linha.

[X] Ctrl + V: Para colar uma linha.

[X] Ctrl + Z: Para apagar uma substring continua.

[X] Ctrl + Shift + seta pra cima: Mover string selecionada pra linha de cima.

[X] Ctrl + Shift + seta pra baixo: Mover string selecionada pra linha de baixo.

[X] Ctrl + seta esquerda: Mover cursor entre substrings pela esquerda.

[X] Ctrl + seta direita: Mover cursor entre substrings pela direita.

[X] Ctrl + Shift + seta esquerda: Remover identação de linha a esquerda.

[X] Ctrl + Shift + seta direita: Identar linha a direita.

[X] Ctrl + A: Selecionar todo o texto.

[X] Ctrl + S: Selecionar uma linha completa.

#### Funcionalidades:

[X] Ao chegar ao final da linha e mover o cursor pra direita, cair para a proxima linha.

[X] Ao chegar ao inicio da linha e mover o cursor para esquerda, subir para a linha anterior.

# Checklist

## Modo de insercao

#### Teclas a adicionar:

[ ] PageUp: Mover cursor ao inicio da linha atual.

[ ] PageDown: Mover cursor ao final da linha atual.

[ ] Home: Mover cursor ao inicio do arquivo.

[ ] End: Mover cursor ao final do arquivo.

#### Atribuicoes de teclas:

[ ] Ctrl + X: Para copiar uma linha.

[ ] Ctrl + BS: Para apagar uma string continua.

[ ] Ctrl + move: Mover o cursor entre strings continuas.

[ ] Ctrl + Enter: Pular linha sem quebrar o texto.

[ ] Ctrl + [: Identar linha a esquerda.

[ ] Ctrl + ]: Identar linha a direita.

[ ] Ctrl + N: Novo arquivo em fila.

[ ] Ctrl + A: Selecionar todo o texto.

[ ] Ctrl + B: Selecionar uma linha completa.

#### Funcionalidades:

[ ] Ao chegar ao final da linha e mover o cursor pra direita, cair para a proxima linha.

[ ] Ao chegar ao inicio da linha e mover o cursor para esquerda, subir para a linha anterior.

## Modo normal

#### Detalhes visuais:

[ ] Mostrar linhas.

[ ] Permitir entradas fora do intervalo ASCII.

#### Comandos:

[ ] Salvar sem sair.

[ ] Editar nome do arquivo.

[ ] Localizar caminho absoluto.

[ ] Remover quebra de linha apos salvar e sair.

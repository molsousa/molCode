# Checklist

## Modo de insercao

#### Teclas a adicionar:

[X] PageUp: Mover cursor ao inicio da linha atual.

[X] PageDown: Mover cursor ao final da linha atual.

[X] Home: Mover cursor ao inicio do arquivo.

[X] End: Mover cursor ao final do arquivo.

#### Atribuicoes de teclas:

[ ] Ctrl + X: Para copiar uma linha.

[ ] Ctrl + V: Para colar uma linha.

[ ] Ctrl + BS: Para apagar uma string continua.

[ ] Ctrl + move: Mover o cursor entre strings continuas.

[ ] Ctrl + Enter: Pular linha sem quebrar o texto.

[ ] Ctrl + [: Identar linha a esquerda.

[ ] Ctrl + ]: Identar linha a direita.

[ ] Ctrl + N: Novo arquivo em fila.

[ ] Ctrl + A: Selecionar todo o texto.

[ ] Ctrl + B: Selecionar uma linha completa.

#### Funcionalidades:

[X] Ao chegar ao final da linha e mover o cursor pra direita, cair para a proxima linha.

[X] Ao chegar ao inicio da linha e mover o cursor para esquerda, subir para a linha anterior.
